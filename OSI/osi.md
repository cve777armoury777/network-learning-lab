# OSI Model

The OSI model explains how network communication works through seven layers.

## Layer 7 - Application
User-facing network services.

Examples:
- HTTP
- HTTPS
- DNS
- SSH

## Layer 6 - Presentation
Data formatting and encryption.

## Layer 5 - Session
Creates and manages communication sessions.

## Layer 4 - Transport
Reliable communication.

Protocols:
- TCP
- UDP

## Layer 3 - Network
Routing and IP addressing.

Devices:
- Router
- Layer 3 Switch

## Layer 2 - Data Link
MAC addressing and switching.

Device:
- Switch

## Layer 1 - Physical
Transmission of bits through:
- Ethernet cables
- Fiber
- Wireless
