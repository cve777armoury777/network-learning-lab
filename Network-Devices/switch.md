# Switch

A switch is a Layer 2 (Data Link Layer) device.

It connects devices inside a local network and forwards frames using MAC addresses.

## Main Functions

- MAC address learning
- Frame forwarding
- VLAN support
- Broadcast handling

## MAC Address Table

Example:

| MAC Address | Port |
|------------|------|
| AA:BB:CC | Port 1 |
| DD:EE:FF | Port 2 |

The switch learns which device is connected to each port.

## Switch Types

### Layer 2 Switch

Uses MAC addresses.

### Layer 3 Switch

Can perform routing using IP addresses.

## Benefits

- Faster communication
- Less unnecessary traffic
- Better network organization
